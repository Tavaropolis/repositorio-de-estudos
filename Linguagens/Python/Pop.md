# pop()
**pop()** é mais uma das funçãode remoção de elementos. Esta função retira o último elemento de uma lista.

```python
konoha = ["Naruto", "Sakura", "Sasuke"]

konoha.pop()

print(konoha) #Sasuke foi retirado da lista por ser o último elemento
```
