# Multiplicação de String

Em Python é possível utilizar a multiplicação **(*)** para repetir uma string determinado número de vezes. Segue exemplo:

```python
text = "Hello World \n" * 5
print(text)

""" resultado na tela:
Hello World 
Hello World 
Hello World 
Hello World 
Hello World
"""
```
