# Concatenação de Tuplas
Assim como [strings](./Str.md) e [listas](./List.md), podemos utilziar o sinal de **+** para juntar duas duplas.

```python
tuple1 = ("a", "b" , "c")
tuple2 = (1, 2, 3)

tuple3 = tuple1 + tuple2
print(tuple3)
```
