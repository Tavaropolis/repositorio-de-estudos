# clear()

O método de Set **clear()** funciona da mesma maneira que o [clear()](./ListClear.md). Ele remove todos os elementos de um set o deixando completamente vazio.

```apex
Set<Integer> numbers = new Set<Integer>{1, 2, 3, 4, 5};
numbers.clear();
```
